🖤 整体风格概览

关键词：「权威感」「极简黑白」「高端金融」「机构信任感」

设计基调：
以黑、白、金、棕为主色调，整体偏向「低饱和度的高级灰黑系」。
给人一种稳健、专业、可信赖的感觉。

视觉氛围：
利用大量留白、细线条和大字号衬线字体营造权威感，
画面不拥挤，信息密度适中。

🎨 颜色体系（Color Palette）
元素	颜色	用法
背景主色	#000000 / #111111	深色背景凸显稳重与高端
辅助背景	#F9F9F9 / #FFFFFF	模块切换时形成对比感
强调色	#C49A6C（古铜金）或 #B9825E	品牌点缀、按钮 hover、标题下划线
字体主色	#1A1A1A / #333333	阅读正文
链接/按钮色	黑底白字 或 白底黑字（反转对比）	

👉 色彩特征：低饱和对比 + 适度金属质感
→ 看起来有“投行范儿”，稳重且有深度。

🔤 字体与排版（Typography）

主字体： 衬线字体（如 Georgia, Times New Roman, Playfair Display）
→ 体现传统与权威感

辅助字体： 无衬线（如 Helvetica Neue / Inter / Source Sans Pro）
→ 用于正文和导航，更现代、易读

排版特点：

标题字距大，留白足

大标题常用 中等粗体 + 首字母大写

行间距充裕（≈1.8–2.0）

模块之间留白明显，呼吸感强

🧱 布局（Layout）

结构： 大模块分区式布局，每个 section 都像一张海报。
→ 一屏讲一个主题，如 “Build with Blackstone” / “Our Impact”。

常用组件：

全屏横幅视频或静态图（含黑色滤镜）+ 白色文字叠层

简洁的两列或三列信息区

按钮采用 “文字 + 圆形箭头” 样式

首页顶部为透明导航栏，滚动后变黑背景

交互细节：

滚动动画平滑

Hover 时元素轻微浮动或加阴影

无繁杂动效，注重「质感」与「平衡感」

🧭 导航与信息层级

顶部导航固定（Sticky Header）

分区清晰：The Firm / Our Clients / What We Do / Insights…

页脚（Footer）信息丰富，含：

联系方式、隐私政策、免责声明

多语言切换

灰黑底 + 白字 + 小字距字体
