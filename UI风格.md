🌈 旅游类网站 UI 规范（高美感 / 大胆色彩版）

⸻

🖤 整体风格概览

关键词：
沉浸式旅行感、自然与城市融合、大幅视觉、轻奢审美、流动感、浪漫氛围、体验导向

设计基调：
高级摄影 + 半透明玻璃拟物 + 大留白 + 大色块跳色点缀

视觉氛围：
	•	首页以 大片风景图 和 色彩叠影 为主
	•	强视觉冲击但不刺眼，整体仍保持高级和现代感
	•	使用大胆饱和色做点缀，使页面“有温度”
	•	整体感觉像 Airbnb Luxe + Lonely Planet + 小红书旅行风 的结合

⸻

🎨 颜色体系（Color Palette）

方案风格：自然系 + 城市光影系 + 跳色强调

元素	颜色	用法
背景主色	#F6F4EF（柔米白）	全站基础背景，让内容干净、照片更突出
辅助背景色	#FBFCFF（浅云蓝）	分区背景、表单底、卡片轻度分离
强调色（Primary）	#FF6B4A（热带珊瑚橙）	CTA 按钮、关键链接、价格标签
次强调色（Secondary）	#4A8FFF（天空蓝）	Hover、标签、高亮导航
跳色（Creative Accent）	#8B4DFF（紫罗兰光）	Banner 叠影、插画、重要组件点缀
字体主色	#2B2B2B（深灰黑）	正文与标题
字体弱色	#6E6E6E	副标题、辅助说明
链接色	#4A8FFF	常规链接、hover 变深

氛围解释
	•	珊瑚橙：给人旅行中的热情与温度
	•	天空蓝：清爽、舒适，让旅游内容不压抑
	•	紫罗兰光：轻科技、轻梦幻，让网站更“有设计感”

⸻

🔤 字体与排版（Typography）

主字体：
	•	中文：HarmonyOS Sans / 思源黑体
	•	英文：Inter / DM Sans（更适合旅行、Lifestyle 风）

辅助字体：
	•	Playfair Display / Cormorant Garamond 用于大标题、Hero 区，使画面更“杂志风”

排版特点：
	•	大号标题（Hero 40–72px）
	•	大段留白
	•	行高偏大（1.5–1.8）
	•	采用 杂志式排版（标题极大+正文极干净）
	•	小圆角（12–16px）+ 半透明背景（用于图片内容）

⸻

🧱 布局（Layout）

结构：
	•	1200–1440 内容区宽
	•	Header 半透明 + 滚动变实色
	•	首页用 Full-width Hero + 三栏 / 两栏混排
	•	大图搭配 浮层文字卡片（模糊玻璃、轻阴影）

常用组件：
	•	旅游目的地卡片（大图+渐变叠层）
	•	笔记/攻略瀑布流（仿小红书）
	•	玻璃拟物浮动按钮（CTA、地图、收藏）
	•	啥地方最热门排行榜
	•	价格卡片带顶部色块
	•	故事式大图标语 Banner
	•	用户评价滑动组件（左右 scroll）

交互细节：
	•	Hover 会提升阴影、轻微缩放
	•	按钮采用 渐变 + 光感 hover
	•	卡片 hover 出现轻微浮层 “查看详情”
	•	大图 Banner 有左右轻动视差效果

⸻

🧭 导航与信息层级

导航栏（Header）：
	•	左侧：Logo（简约线条旅行 icon）
	•	右侧：Explore / 目的地 / 视频 / 广告合作 / Blog / Login
	•	右侧最后一个按钮为 醒目的 CTA：开始规划旅程（珊瑚橙）
	•	滚动后变成纯白背景 + 阴影，保持清晰度
	•	大屏下二级导航可采用 横向 tabs（城市 / 海岛 / 主题旅拍 / 套票）
